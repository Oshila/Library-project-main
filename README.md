# Library Book Management System

**Python Software Development Project**  
**Course: ICT323**  
**Bells University of Technology – New Horizons ICT**  
**300 Level – Department of Computer Science**

Desktop application with **Tkinter GUI** for managing library books, students, and librarians.  
Uses CSV files for offline storage.

## Features
- Student & Librarian login/registration
- View all books
- Borrow & return books
- Add & remove books (Librarian only)
- Modern Tkinter graphical interface
- Persistent data using CSV files
- Offline desktop app

### Tools Used

- Python 3
- Tkinter (GUI)
- CSV file handling

## How to Run the Project

1. Install Python 3 (if not already installed)
2. Open the project folder
3. Run the command: Python main.py

## Limitations

- Runs offline only
- Uses CSV files instead of a full database
- Basic GUI (can be improved with more advanced styling or libraries)
